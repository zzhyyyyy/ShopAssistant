`const`在声明时必须初始化，否则会导致编译错误
`constexpr`(c++11)引入，可以在文件编译阶段就完成计算
```
constexpr int add(int a,int b)
{
	return a+b;
}
int main()
{
	const int a = add(3,5);//add参数3,5都是字面量。故在编译阶段直接将add(3,5)替换至常量8
	int b = add(a,1);//a是常量，1是字面量。故在编译阶段直接将add(a,1)替换至常量9
	int c = add(a,b);//a是常量 但b是非常量，必须进入add函数处理，无法在编译阶段完成计算
}
```

### 使用`constexpr`做函数时要求
1.返回值和函数传参必须是字面类型(常量)，函数内的中间临时变量可以是非常量的
2.函数内部不能调用其他非常值函数
3.不能抛异常
